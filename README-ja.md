# Web Site Game
Web Site Gameはオンライン、オフラインでも動作するhtmlとjavascript、CSSをメインとしたWebゲームを作成しています

# 起動方法
[こちら](https://github.com/Hiryuto/website-game/releases/latest)から最新のリリースにアクセスしソースコードをダウンロードして下さい

その後index.htmlを表示するとプレイすることができます
